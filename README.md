# West Beach Company - Static Website

موقع HTML/CSS/JS خالص جاهز للنشر على GitHub Pages بدون أي بناء.

## النشر على GitHub Pages
1. أنشئ مستودع جديد على GitHub (Public).
2. ارفع **محتويات** هذا المجلد كاملة إلى **جذر المستودع** (ليس داخل مجلد فرعي):
   - index.html, home.html, products.html, contact.html
   - style.css, script.js
   - images/ (المجلد كاملاً)
   - .nojekyll
3. Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / `/ (root)` → Save.
4. انتظر دقيقة وافتح: `https://<username>.github.io/<repo-name>/`

لا تحتاج Node ولا Bun ولا أي build — كل شيء يعمل مباشرة في المتصفح.
